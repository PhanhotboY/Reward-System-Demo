# Reward System Demo

### Requirements

- WSL2 (Windows Sub-system for Linux)
- Truffle Ganache
- Docker Desktop

### Setup

1. Create new Ganache workspace, set gas limit to `30000000`, select Hostname: `vEthernet (WSL)`

2. Enable yarn (optional):

```bash
corepack enable
```

3. Compose

```bash
docker compose up -d
```
